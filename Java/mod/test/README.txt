[使用前に]
マイクラにMODとして入れる場合はPotatoMod-1.16.5-1.0.0.jarを使ってください。
(マイクラバージョン1.16.5のForge用です。)
実際に使ってテストしたので競合でもなければバグらないと思います。
その他のフォルダやファイルはMODを作成するためのものです。

[MODについて]
MOD開発のチュートリアル的なやつで作ったチートMODです...w
土から「新アイテム」が作られ、新アイテムから「新ブロック」が作れます。
新ブロックに特に使い道はないですが、貯蓄用に圧縮する目的ならまあぎりぎり使えます()
新アイテムをかまどで燃やすと、一瞬でダイアモンド1つと10万経験値が得られます。
10万経験値は大体レベル0がレベル150になるくらいの量です。
あと、新ブロックは石以上のピッケルで掘ってください。

[詳細]
IDについて
「新アイテム」はtest:newitem
「新ブロック」はtest:newblock
